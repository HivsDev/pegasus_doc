--- # Hi3403-v1.0-beta1 The first beta provides only baseline functionality. Refer to the docs in
`docs/` and the examples in `sample/` to get started. - Supports Hi3403V100 /
- Supports Linux 6.6.86 with both Clang and GCC toolchains
- Audio/video capture (with ISP processing), audio/video encode/decode, audio/video output, video pre-processing, dpico-AI; sample programs exercise each
- Adapts the OpenHarmony 5.1.0 Release patchset to Hi3403V100 / , enables OpenHarmony Small-system to run media + graphics baseline features, supports XTS certification, and runs the MPP samples
