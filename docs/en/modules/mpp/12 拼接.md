--- # Stitching

## Overview AVS (Any View Stitching) implements the following functions: panorama stitching of multiple video streams and output of images according to a specified projection mode. The entire stitching process mainly includes calibration and stitching output. This chapter primarily describes the stitching output part. In addition, AVS also supports horizontal, vertical, or tiled non-blended stitching. > **Note:**[¶](#overview-avs-any-view-stitching-implements-the-following-functions-panorama-stitching-of-multiple-video-streams-and-output-of-images-according-to-a-specified-projection-mode-the-entire-stitching-process-mainly-includes-calibration-and-stitching-output-this-chapter-primarily-describes-the-stitching-output-part-in-addition-avs-also-supports-horizontal-vertical-or-tiled-non-blended-stitching-note "锚链接")

<a name="ZH-CN_TOPIC_0000002408258334"></a><a name="ZH-CN_TOPIC_0000002441697577"></a>> This chapter only supports the Hi3403V100 module. ## Functional Description ### Basic Concepts - **GROUP** AVS provides the concept of a group to users. The maximum number of available groups is [OT\_AVS\_MAX\_GRP\_NUM](#ZH-CN_TOPIC_0000002408258334). Each GROUP time-shares the AVS hardware. Each AVS GROUP contains multiple PIPEs and multiple channels. - **PIPE** The PIPE of an AVS group. Used for inputting stitching source images. The number of PIPEs equals the number of stitching streams. Users can bind to front-end through the system or send images to PIPE for stitching. - **CHANNEL** The channel of an AVS group. Used for outputting the stitched result image. Channel 0 is the original stitched output image. Other channels are reduced versions of channel 0, with a maximum reduction factor of 15x. - **Projection** Projects the panoramic spatial image onto planar coordinates for encoding and transmission. AVS supports four projection modes: Equirectangular, Cylindrical, Rectilinear, and Cube map. ## API Reference This functional module provides the following MPIs to the user: - [ss\_mpi\_avs\_create\_grp](#ZH-CN_TOPIC_0000002441697577): Creates an AVS GROUP.
