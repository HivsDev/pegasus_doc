## Overview 3A statistical information and related configurations provided by ISP. ## Function Description ### AE Statistics Implements the statistics for auto exposure information. The software adjusts the sensor based on the statistical information to achieve auto exposure functionality. AE statistics include zone-based R/Gr/Gb/B mean statistics, weighted global R/Gr/Gb/B means, and 1024-bin histogram information. #### AE Zone Statistics AE supports a maximum of 17x15 blocks and a minimum of 1x1 block. Each block can output 16-bit R, Gr, Gb, B mean values (Average R/Gr/Gb/B). Taking R and Gr as examples, the correspondence between block statistics read addresses and data is shown in [Table 1](#_Ref430175021). m represents the number of horizontal zones, n represents the number of vertical zones. **Table 1** AE Zone Statistics [¶](#overview-3a-statistical-information-and-related-configurations-provided-by-isp-function-description-ae-statistics-implements-the-statistics-for-auto-exposure-information-the-software-adjusts-the-sensor-based-on-the-statistical-information-to-achieve-auto-exposure-functionality-ae-statistics-include-zone-based-rgrgbb-mean-statistics-weighted-global-rgrgbb-means-and-1024-bin-histogram-information-ae-zone-statistics-ae-supports-a-maximum-of-17x15-blocks-and-a-minimum-of-1x1-block-each-block-can-output-16-bit-r-gr-gb-b-mean-values-average-rgrgbb-taking-r-and-gr-as-examples-the-correspondence-between-block-statistics-read-addresses-and-data-is-shown-in-table-1-m-represents-the-number-of-horizontal-zones-n-represents-the-number-of-vertical-zones-table-1-ae-zone-statistics "锚链接")

#### AE Histogram Statistics AE provides 1024-bin histogram statistics. The bit width of each histogram bin is 25 bits. The histogram range is [0, 1023] in 10-bit precision. #### AWB Statistics Implements statistics for auto white balance information.
