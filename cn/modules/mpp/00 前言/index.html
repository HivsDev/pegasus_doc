    前言¶

    概述

    本文为使用MPP媒体处理芯片进行开发的程序员而写，目的是供您在开发过程中查阅媒体处理软件开发包的各种参考信息，包括API、头文件、错误码等。

    本文档描述MPP媒体处理软件的各个API的使用方法，以及相关的数据结构和错误码。

    说明： 未有特殊说明，SS927V100与Hi3403V100内容完全一致。

    产品版本

    与本文档相对应的产品版本如下。

    产品版本

    读者对象

    本文档主要适用于以下工程师：

    • 技术支持工程师
    • 软件开发工程师

    文档章节

    本文档集按以下章节组织：

    • ISP 图像处理器 — 图像信号处理开发参考、图像调优指南、颜色调优说明及 FAQ
    • 视频处理子系统 — HDMI/MIPI 开发参考、PCIE 级联应用指南、拼接 FAQ
    • IVE/IVS 智能视觉 — IVE 和 IVS API 参考
    • 媒体处理 (MPP) — MPP 媒体处理软件开发参考（含视频输入/输出/编码/解码、音频、区域管理、Proc 调试等）
    • 智能分析引擎 — SVP2.0 开发指南、ATC/AMCT 工具使用指南、DPU/Profiling 工具等
    • 芯片参考 API — CIPHER、GFBG、KLAD、OTP、TDE、音频组件 API 参考
