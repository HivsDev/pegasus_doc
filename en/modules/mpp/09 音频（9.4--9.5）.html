    09 · Audio 9.4–9.5¶

    title: "Data Types" source: /sessions/sharp-sweet-allen/mnt/hi3403-build/pegasus/docs/zh-CN/MPP 媒体处理软件 V5.0 开发参考/09 音频（9.4--9.5）.md --- # Audio ## Data Types

    Audio Input and Output The data types and data structures related to audio input and output are defined as follows: - OT_AIO_MAX_NUM: Defines the maximum number of audio input/output devices.¶¶

    • OT_AI_DEV_MAX_NUM: Defines the maximum number of audio input devices.
    • OT_AO_DEV_MAX_NUM: Defines the maximum number of audio output devices.
    • OT_AIO_MAX_CHN_NUM: Defines the maximum number of channels for audio input/output devices.
    • OT_AI_MAX_CHN_NUM: Defines the maximum number of channels for audio input devices.
    • OT_AO_MAX_CHN_NUM: Defines the maximum number of channels for audio output devices.
    • OT_AO_SYS_CHN_ID: Defines the system tone channel ID for audio output devices.
    • OT_AI_TALKVQE_MASK_HPF: Mask for the Talk VQE HPF function.
    • OT_AI_TALKVQE_MASK_AEC: Mask for the Talk VQE AEC function.
    • OT_AI_TALKVQE_MASK_AGC: Mask for the Talk VQE AGC function.
    • OT_AI_TALKVQE_MASK_EQ: Mask for the Talk VQE EQ function.
    • OT_AI_TALKVQE_MASK_ANR: Mask for the Talk VQE ANR function.
    • OT_AI_RECORDVQE_MASK_HPF: Mask for the Record VQE HPF function.
    • OT_AI_RECORDVQE_MASK_RNR: Mask for the Record VQE RNR function.
    • OT_AI_RECORDVQE_MASK_HDR: Mask for the Record VQE HDR function.
    • OT_AI_RECORDVQE_MASK_DRC: Mask for the Record VQE DRC function.
    • OT_AI_RECORDVQE_MASK_EQ: Mask for the Record VQE EQ function.
    • OT_AI_RECORDVQE_MASK_AGC: Mask for the Record VQE AGC function.
    • OT_AI_TALKVQEV2_MASK_PNR: Defines the Mask for the Talk VqeV2 PNR function.
    • OT_AI_TALKVQEV2_MASK_NR: Defines the Mask for the Talk VqeV2 NR function.
    • OT_AI_TALKVQEV2_MASK_EQ: Defines the Mask for the Talk VqeV2 EQ function.
    • OT_AI_TALKVQEV2_MASK_AGC: Defines the Mask for the Talk VqeV2 AGC function.
    • OT_AI_TALKVQEV2_MASK_DRC: Defines the Mask for the Talk VqeV2 DRC function.
    • OT_AI_TALKVQEV2_MASK_LIMITER: Defines the Mask for the Talk VqeV2 LIMITER function.
    • OT_AI_TALKVQEV2_MASK_DEREVERB: Defines the Mask for the Talk VqeV2 DEREVERB function.
    • OT_AI_TALKVQEV2_MASK_FMP: Defines the Mask for the Talk VqeV2 FMP function.
    • OT_AI_TALKVQEV2_MASK_WNR: Defines the Mask for the Talk VqeV2 WNR function.
    • OT_AI_TALKVQEV2_MASK_AEC: Defines the Mask for the Talk VqeV2 AEC function.
    • OT_AI_TALKVQEV2_MASK_HS: Defines the Mask for the Talk VqeV2 HS function.
    • OT_AO_VQE_MASK_HPF: Mask for the AO VQE HPF function.
    • OT_AO_VQE_MASK_ANR: Mask for the AO VQE ANR function.
    • OT_AO_VQE_MASK_AGC: Mask for the AO VQE AGC function.
    • OT_AO_VQE_MASK_EQ: Mask for the AO VQE EQ function.
    • OT_MAX_AUDIO_FILE_PATH_LEN: Maximum length limit for the audio file save path.
    • OT_MAX_AUDIO_FILE_NAME_LEN: Maximum length limit for the audio file save name.
    • OT_AUDIO_FRAME_CHN_NUM: Defines the maximum number of channels for storing audio data.
    • OT_VQE_EQ_BAND_NUM: Defines the number of adjustable EQ frequency bands.
    • OT_VQE_DRC_SEC_NUM: Defines the configurable number of DRC dynamic curve segments.
    • OT_AEC_ERL_BAND_NUM: Defines the number of AEC ERL protection frequency bands.
    • OT_AEC_ERL_BAND_VALUE_NUM: Defines the number of AEC ERL protection frequency band values.
    • OT_TALKVQEV2_EQ_BAND_NUM: Defines the number of adjustable EQ frequency bands for Talk VqeV2.
    • ot_audio_clk_select: Defines the audio clock source.
    • ot_audio_sample_rate: Defines the audio sample rate.
    • ot_audio_bit_width: Defines the audio sample bit width.
    • ot_aio_mode: Defines the audio input/output working mode.
    • ot_aio_i2s_type: Defines the device I2S interface type.
    • ot_audio_snd_mode: Defines the audio channel mode.
    • ot_audio_mod_param: Defines the audio module parameter structure.
    • ot_aio_attr: Defines the audio input/output device attribute structure.
    • ot_ai_chn_param: Defines the channel parameter structure.
    • ot_audio_frame: Defines the audio frame data structure.
    • ot_aec_frame: Defines the acoustic echo cancellation reference frame information structure.
    • ot_audio_agc_cfg: Defines the audio automatic gain control configuration information structure.
    • ot_ai_aec_cfg: Defines the audio acoustic echo cancellation configuration information structure.
    • ot_audio_anr_cfg: Defines the audio speech noise reduction function configuration information structure.
    • ot_audio_hpf_freq: Defines the audio high-pass filter cutoff frequency.
    • ot_audio_hpf_cfg: Defines the audio high-pass filter cutoff frequency configuration.
    • ot_ai_rnr_cfg: Defines the audio recording noise reduction function configuration information structure.
    • ot_vqe_work_state: Defines the audio VQE working mode.
    • ot_vqe_record_type: Defines the recording type.
    • ot_audio_eq_cfg: Defines the audio equalizer function configuration information structure.
    • ot_ai_hdr_cfg: Defines the audio high dynamic range function configuration information structure.
    • ot_ai_drc_cfg: Defines the audio dynamic range control function configuration information structure.
    • ot_ai_talk_v2_pnr_cfg: Defines the audio Talk VqeV2 secondary noise reduction function configuration information structure.
    • ot_ai_talk_v2_nr_cfg: Defines the audio Talk VqeV2 noise reduction function configuration information structure.
    • ot_ai_talk_v2_agc_cfg: Defines the audio Talk VqeV2 automatic gain control function configuration information structure.
    • ot_ai_talk_v2_eq_cfg: Defines the audio Talk VqeV2 equalizer function configuration information structure.
    • ot_ai_talk_v2_aec_cfg: Defines the audio Talk VqeV2 acoustic echo cancellation function configuration information structure.
    • ot_ai_talk_v2_wnr_cfg: Defines the audio Talk VqeV2 wind noise reduction function configuration information structure.
    • ot_ai_talk_v2_fmp_cfg: Defines the audio Talk VqeV2 parallel processing function configuration information structure.
    • ot_ai_talk_v2_hs_cfg: Defines the audio Talk VqeV2 howling suppression function configuration information structure.
    • ot_ai_talk_vqe_cfg: Defines the audio input sound quality enhancement (Talk) configuration information structure.
    • ot_ai_record_vqe_cfg: Defines the audio input sound quality enhancement (Record) configuration information structure.
    • ot_ai_talk_vqe_v2_cfg: Defines the audio input sound quality enhancement (Talk V2) configuration information structure.
    • ot_ao_vqe_cfg: Defines the audio output sound quality enhancement configuration information structure.
    • ot_vqe_mod_cfg: Defines the sound quality enhancement and resampling module configuration information structure.
    • ot_audio_vqe_register: Defines the sound quality enhancement and resampling module registration structure.
    • ot_audio_stream: Defines the audio stream structure.
    • ot_ao_chn_state: Data buffer status structure for audio output channels.
    • ot_audio_track_mode: Audio device channel mode type.
    • ot_audio_fade_rate: Audio device fade-in/fade-out rate type.
    • ot_audio_fade: Audio device fade-in/fade-out setting structure.
    • ot_g726_bps: Defines the G.726 codec bit rate.
    • ot_adpcm_type: Defines the ADPCM codec type.
    • ot_audio_save_file_info: Defines the audio file saving function configuration information structure.
    • ot_audio_file_status: Defines the audio file saving status structure.
    • ot_ai_chn_mode: Defines the AI channel working mode.
    • ot_ai_chn_attr: Defines the AI channel attribute structure. Please refer to the source document for detailed type definitions and member descriptions. The content below documents the complete set of audio data types and error codes for the MPP audio module.
