Parameter Description

    Parameter Description
    vpss module param1 split_node_num Number of split nodes. valid range: [1, 16]. Default value: 3. Modified via the module parameter interface.
    coverex_ratio_support Whether Coverex relative coordinates are supported. Y: Enabled; N: Disabled.
    schedule_mode Scheduling mode, set via the SYS interface. NORMAL: Normal scheduling mode. QUICK: Quick scheduling mode.
    nr_low_delay_en 3DNR delay mode enabled. Y: Enabled; N: Disabled.
    multi_task Group multi-task enabled. Y: Enabled; N: Disabled.
    high_profile VPSS module high-performance switch, only supported by Hi3403V100. N: Not enabled; Y: Enabled.
    weight_threshold VPSS group chain threshold. valid range: [40, 160].
    vpss module param2 output_low_delay_support VPSS output low-latency switch supported. Y: Enabled; N: Not enabled.
    cal_cost_time_en VPSS proc statistical processing time data switch. Y: Enabled; N: Not enabled. When disabled, statistics such as cost_time, max_cost_time, proc_time, and interrupt work time in VPSS proc are not collected.
    vpss grp attr1 VPSS Grp attribute 1 grp Grp ID. valid range: [0,OT_VPSS_MAX_GRP_NUM)
    max_w Maximum input image width for the group. Unit: pixel.
    max_h Maximum input image height for the group. Unit: pixel.
    pixel_fmt Input image pixel format for the group.
    src_rate GRP source frame rate.
    dst_rate GRP destination frame rate.
    user_ctrl User frame rate control. Y: On; N: Disabled.
    backup Backup enabled status. Displays N when no frame input at the beginning, and shows actual status based on the situation when frames are available. Y: On; N: Disabled.
    nr_type 3DNR mode. VIDEO: Video mode; SNAP: Snapshot mode. VIDEO_SPA: Video pure spatial mode.
    nr_ref_cmp Reference frame compression mode. Y: Compressed; N: Not compressed.
    nr_motion_mode 3DNR motion vector mode. NORM: Normal mode; COMPENSATION: Motion compensation mode.
    vpss grp attr2 VPSS Grp attribute 2 grp Grp ID. valid range: [0,OT_VPSS_MAX_GRP_NUM)
    buf_share_en User-configured input/output buffer sharing status. Y: On; N: Disabled.
    buf_share_chn User-configured input/output buffer sharing channel number. valid range: [0, OT_VPSS_MAX_PHYS_CHN_NUM)
    dci DCI enabled status Y: On; N: Disabled.
    dei_mode DEI mode. OFF: Force off; ON: Force on; AUTO: Adaptive.
    dei_width DEI maximum processing width. valid range: [64, OT_VPSS_MAX_DEI_HEIGHT]
    dei_height DEI maximum processing height. valid range: [64, OT_VPSS_MAX_IMG_HEIGHT]
    mcf_en MCF enabled. Y: Enabled; N: Disabled.
    act_buf_share Actual effective input/output buffer sharing status. X: Multiplexed channel number, valid range: [0, OT_VPSS_MAX_PHYS_CHN_NUM) N: Disabled.
    vpss grp param VPSS group parameters and configuration information grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    contrast DCI intensity.
    ie_strength IE intensity.
    sizer Whether size filtering is enabled.
    sizer_width Filter size width.
    sizer_height Filter size height.
    mosaic Group mosaic count.
    cover Group cover count.
    overlay Group OSD count.
    luma Luminance and task count last submitted to the group.
    vpss grp crop info VPSS group CROP information grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    crop_en Whether CROP function is enabled. N: Disabled; Y: On.
    coord Coordinate type. RAT: Relative coordinates; ABS: Absolute coordinates.
    width Width of the CROP rectangle, cannot exceed the maximum image width.
    height Height of the CROP rectangle, cannot exceed the maximum image height.
    trim_x Actual image starting point coordinate.
    trim_y Actual image starting point coordinate.
    trim_width Actual image width.
    trim_height Actual image height.
    vpss 3dnr x param VPSS 3DNR V1 standard parameters grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    version 3DNR standard parameter version VER_1: Version 1; VER_2: Version 2.
    sf_strength Luminance spatial denoising intensity (standard parameter)
    tf_strength Luminance temporal denoising intensity (standard parameter)
    csf_strength Chroma spatial denoising intensity (standard parameter)
    ctf_strength Chroma temporal denoising intensity (standard parameter)
    cvbs_strength Intensity of the ability to handle luminance crosstalk on input (standard parameter)
    vpss 3dnr x param VPSS 3DNR V2 standard parameters grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    version 3DNR standard parameter version. VER_1: Version 1; VER_2: Version 2.
    opt_mode 3DNR working mode.
    iso ISO value.
    ref0_en 3DNR level 0 temporal switch.
    ref1_en 3DNR level 1 temporal switch. Debugging is generally not recommended; it is advised to keep the default value of 1.
    nry0_en, nry1_en, nry2_en, nry3_en 4-level luminance filter enabled flag. 0: Current filter not enabled; 1: Current filter enabled.
    nrc0_en, nrc1_en 2-level chroma filter enabled flag. 0: Current filter not enabled; 1: Current filter enabled.
    limit_range_en Limit range mode switch. 0: Not enabled; 1: Enabled.
    sfc Spatial filter strength.
    auto_math Adjusts chroma temporal filter parameters.
    sfs2_0, sfs4_0, sfn_0 Adjusts level 0 spatial filter parameters.
    tfs0_0, math_0 Adjusts level 0 temporal filter parameters.
    sfs2_1, sfs4_1, sfn_1 Adjusts level 1 spatial filter parameters.
    tfs1_1, math0_1, math1_1 Adjusts level 1 temporal filter parameters.
    sfs2_2, sfs4_2, sfn_2 Adjusts level 2 spatial filter parameters.
    tfs0_2, math0_2 Adjusts level 2 temporal filter parameters.
    sfs2_, sfs4_3, sfn_3, sfn4 Adjusts level 3 spatial filter parameters.
    sfs1_c, sfs2_c, sfs4_c, sfn0_c, sfn1_c Adjusts chroma level 1 spatial parameters.
    vpss 3dnr adv param VPSS 3DNR advanced parameters grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    mdz Region motion threshold (advanced parameter)
    hard_tf_strength Strong denoising channel temporal/spatial blending ratio (advanced parameter)
    edz Region strong edge threshold (advanced parameter)
    weak_tf_strength Weak denoising channel temporal/spatial blending ratio (advanced parameter)
    vpss chn crop info Channel crop information grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    chn Channel ID. valid range: [0, OT_VPSS_MAX_ CHN_NUM)

    coord Coordinate type. RAT: Relative coordinates; ABS: Absolute coordinates

    trim_width Actual image width.
    trim_height Actual image height.
    vpss ldci attr VPSS LDCI attributes grp Grp ID. valid range: [0, OT_VPSS_MAX_GRP_NUM)
    enable Whether LDCI function is enabled. Y: On. N: Disabled. This status reflects the actual effective state, not the state configured through the interface.
    color_range Quantization range of the input YUV. LIMITED: YUV image quantization range is [16, 235]. FULL: YUV image quantization range is [0, 255].
    blc_ctrl Dark area gain control intensity, used to suppress noise in dark areas.
    he_pos_weight The weight curve for blending the local histogram equalization result with the original image. It controls the brightness enhancement level based on luminance. The curve follows a GAUSS distribution. This parameter is the maximum value of the GAUSS distribution.
    he_pos_sigma The weight curve for blending the local histogram equalization result with the original image. It controls the brightness enhancement level based on luminance. The curve follows a GAUSS distribution. This parameter is the mean square error of the GAUSS distribution.
    he_pos_mean The weight curve for blending the local histogram equalization result with the original image. It controls the brightness enhancement level based on luminance. The curve follows a GAUSS distribution. This parameter is the mathematical expectation of the GAUSS distribution.
    he_neg_weight The weight curve for blending the local histogram equalization result with the original image. It controls the dimming level based on luminance. The curve follows a GAUSS distribution. This parameter is the maximum value of the GAUSS distribution.
    he_neg_sigma The weight curve for blending the local histogram equalization result with the original image. It controls the dimming level based on luminance. The curve follows a GAUSS distribution. This parameter is the mean square error of the GAUSS distribution.
    he_neg_mean The weight curve for blending the local histogram equalization result with the original image. It controls the dimming level based on luminance. The curve follows a GAUSS distribution. This parameter is the mathematical expectation of the GAUSS distribution.
