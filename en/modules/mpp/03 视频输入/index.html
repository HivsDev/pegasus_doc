The user must set the appropriate mask configuration based on the actual connection. The most significant bit of the mask corresponds to pin D15. For example, if a 10-bit input sensor is connected to pins D6~D15, the mask is configured as 0x FFC00000. Similarly, for 14-bit input, the corresponding mask is configured as 0x FFFC0000. The VI data line order is from low to high.
