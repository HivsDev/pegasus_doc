    Overview 3A statistical information and related configurations provided by ISP. ## Function Description ### AE Statistics Implements the statistics for auto exposure information. The software adjusts the sensor based on the statistical information to achieve auto exposure functionality. AE statistics include zone-based R/Gr/Gb/B mean statistics, weighted global R/Gr/Gb/B means, and 1024-bin histogram information. #### AE Zone Statistics AE supports a maximum of 17x15 blocks and a minimum of 1x1 block. Each block can output 16-bit R, Gr, Gb, B mean values (Average R/Gr/Gb/B). Taking R and Gr as examples, the correspondence between block statistics read addresses and data is shown in Table 1. m represents the number of horizontal zones, n represents the number of vertical zones. Table 1 AE Zone Statistics ¶¶

    Memory address Zone Memory Read Data
    [31:16] [15:0]
    base + 0x0 Zone(0, 0) R_avg[15:0] Gr_avg[15:0]
    base + 0x4 Zone(1, 0) R_avg[15:0] Gr_avg[15:0]

    #### AE Histogram Statistics AE provides 1024-bin histogram statistics. The bit width of each histogram bin is 25 bits. The histogram range is [0, 1023] in 10-bit precision. #### AWB Statistics Implements statistics for auto white balance information. The software adjusts the white balance gain based on the statistical information to achieve auto white balance functionality. #### AF Statistics Implements statistics for auto focus information, providing high-frequency information of the image windows for focus evaluation. ## API Reference - ss_mpi_isp_query_ae_stats: Query AE 1024-bin histogram statistics and four-component statistics. - ss_mpi_isp_query_awb_stats: Query AWB statistics. - ss_mpi_isp_query_af_stats: Query AF statistics. - ss_mpi_isp_set_ae_stats_attr: Set AE statistics attributes. - ss_mpi_isp_get_ae_stats_attr: Get AE statistics attributes. - ss_mpi_isp_set_awb_stats_attr: Set AWB statistics attributes. - ss_mpi_isp_get_awb_stats_attr: Get AWB statistics attributes. - ss_mpi_isp_set_af_stats_attr: Set AF statistics attributes. - ss_mpi_isp_get_af_stats_attr: Get AF statistics attributes. ## Data Types - OT_ISP_AE_ZONE_ROW: AE zone row count. - OT_ISP_AE_ZONE_COL: AE zone column count. - ot_isp_ae_stats_attr: Defines AE statistics attributes. - ot_isp_awb_stats_attr: Defines AWB statistics attributes. - ot_isp_af_stats_attr: Defines AF statistics attributes. - ot_isp_ae_stats: Defines AE statistics information. - ot_isp_awb_stats: Defines AWB statistics information. - ot_isp_af_stats: Defines AF statistics information. ## Overview Additional ISP configuration and algorithm development reference. ## Function Description Describes the ISP pipeline configuration, sensor integration, and algorithm registration workflows. ## API Reference - ss_mpi_isp_init: Initialize ISP. - ss_mpi_isp_exit: Exit ISP. - ss_mpi_isp_run: Start ISP running. - ss_mpi_isp_stop: Stop ISP. ## ISP ISP pipeline configuration and management. ### Overview The ISP pipeline processes raw Bayer data through a series of image processing modules to produce final image output. This section describes the ISP configuration and control interfaces. ### Function Description ISP functional modules and their configuration methods. ### Software Flow Describes the ISP software initialization and configuration flow. ### Sensor Integration Describes how to integrate and configure image sensors with the ISP. ### Overview Sensor driver development and integration reference for connecting sensors to the ISP pipeline. ### AE Algorithm Registration with ISP Library Describes the process of registering the AE algorithm library with the ISP library, including the registration interfaces and callback functions required for integration. ### AWB Algorithm Registration with ISP Library Describes the process of registering the AWB algorithm library with the ISP library, including the registration interfaces and callback functions required for integration. ### Developing Custom AF Algorithm Describes the development process for user-defined auto focus algorithms, including the interface specifications and integration requirements. ### AF Statistics Usage Instructions Provides detailed instructions on how to use AF statistical information for focus evaluation and auto focus algorithm development. ### No-Photosensor AE Fast Convergence Mode (No-Photosensor Quick Start) Usage Instructions Describes the usage of the AE fast convergence mode without a photosensor, enabling quick AE convergence during system startup.
