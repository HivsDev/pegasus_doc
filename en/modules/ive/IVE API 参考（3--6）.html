Fixed-point data types: - ot_svp_data: Defines 2D image information in bytes. - ot_svp_src_data: Defines 2D source data information in bytes. - ot_svp_dst_data: Defines 2D output data information in bytes. - ot_svp_8bit: Defines an 8-bit data union. - ot_svp_point_u16: Defines a u16-bit point information structure. - ot_svp_point_s16: Defines an s16-bit point information structure. - ot_svp_point_s25q7: Defines a point information structure represented by s25q7. - ot_svp_point_u14q2: Defines a point information structure represented by u14q2. - ot_svp_rect_u32: Defines a rectangle information structure represented by u32. - ot_svp_rect_u16: Defines a rectangle information structure represented by u16. - ot_svp_rect_s24q8: Defines a rectangle information structure represented by s24q8. - ot_svp_lut: Defines a lookup table structure. IVE-related data types and data structures are defined as follows: - ot_ive_handle: Defines the IVE handle. - OT_IVE_HIST_NUM: Defines the number of histogram bins. - OT_IVE_MAP_NUM: Defines the number of mapping lookup table entries. - OT_IVE_MAX_RGN_NUM: Defines the maximum number of connected regions. - OT_IVE_ST_MAX_CORNER_NUM: Defines the maximum number of Shi-Tomasi-like corners. - OT_IVE_MASK_NUM: Mask array length. - OT_IVE_ARR_RESERVED_NUM_TWO: Reserved field array length 2. - OT_IVE_ARR_RESERVED_NUM_THREE: Reserved field array length 3. - OT_IVE_ARR_RESERVED_NUM_EIGHT: Reserved field array length 8. - OT_IVE_ARR_RESERVED_NUM_TWELVE: Reserved field array length 12. - OT_IVE_ARR_RESERVED_NUM_FOURTEEN: Reserved field array length 14. - ot_ive_err_code: Defines error codes. - ot_ive_dma_mode: Defines DMA operation mode. - ot_ive_dma_ctrl: Defines DMA control information. - ot_ive_filter_ctrl: Defines template filter control information. - ot_ive_csc_mode: Defines color space conversion mode. - ot_ive_csc_ctrl: Defines color space conversion control information. - ot_ive_filter_and_csc_ctrl: Defines composite template filter plus color space conversion control information. - ot_ive_sobel_out_ctrl: Defines sobel output control information. - ot_ive_sobel_ctrl: Defines sobel edge extraction control information. - ot_ive_mag_and_ang_out_ctrl: Defines the output format for canny edge magnitude and angle calculation. - ot_ive_mag_and_ang_ctrl: Defines control information for canny edge magnitude and angle calculation. - ot_ive_dilate_ctrl: Defines dilation control information. - ot_ive_erode_ctrl: Defines erosion control information. - ot_ive_threshold_mode: Defines image binarization output format. - ot_ive_threshold_ctrl: Defines image binarization control information. - ot_ive_sub_mode: Defines the output format for image subtraction. - ot_ive_sub_ctrl: Defines control parameters for image subtraction. - ot_ive_integ_out_ctrl: Defines integral image output control parameters. - ot_ive_integ_ctrl: Defines integral image calculation control parameters. -
